# How Software Works

Interactive online course that explains how software works from binary to production, structured as a multi-module web book with MDX content and syntax-highlighted code examples.

## What It Does

- Structured course covering: build systems, machine-level computing, operating systems, compilers, networking, databases, web frameworks, security, and DevOps
- MDX-based content (Markdown with embedded React components)
- Syntax highlighting via Shiki
- Sidebar navigation with expandable modules and chapter tracking
- Responsive dark-themed reading experience

## Tech Stack

- React 19 + React Router v7
- MDX (via @mdx-js/rollup) for content authoring
- Shiki for syntax highlighting
- Tailwind CSS v4
- Vite, TypeScript

## Getting Started

```bash
npm install
npm run dev       # Start dev server
npm run build     # Production build
npm run preview   # Preview production build
```
